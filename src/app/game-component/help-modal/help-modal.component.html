Help

Marvel Champions Cardle is an wordle-like game where you have to guess a random Marvel Champions player card each day.

How do I play?

To play simply enter a card name into the search bar. This will search for cards that contain the name and show those under the input field. If you click on a card it will add the card as a guess.
This will add a new card info element under the search bar: Each value of the guessed card will be compared to the correct card:
  • Green text shows that the value is correct.
  • Orange text shows that the value is partially correct.
  • Red-ish text shows that correct value is higher.
  • Blue text shows that correct value is lower.
You can also view this info in the legend. Above the search bar is a summarized info of the correct card and what you have guessed so far: Values marked as ??? have not been guessed yet.
You can click on the correct/partially correct values to add filters to your search. Filtered values are marked with an underscore and also shown in the search bar.

If you guessed the correct card, it will open a success popup and you can share your guesses with your friends (to brag of course). You can reopen the popup by pressing the "Show result" button.

Other Parameters

Additionally you can:
  • Play different past days by selecting the date. Days that are marked green have been solved. Days that are marked orange have been guessed but not solved.
  • Change the language to German. This will change every occurrence of card names to the german version of the name. If the german name doesn't exist (ie insufficient data or not translated), it will fallback to the english name.
You can show this help anytime by pressing help in the navigation (on mobile you might need to press the menu button first).
